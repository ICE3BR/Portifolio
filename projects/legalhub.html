Legal-Hub 🔒 Privado

Central desktop de automações jurídicas construída com PyWebView + Python + HTML/CSS/JS. Autenticação LDAP corporativa e 6 módulos ativos em produção, utilizados diariamente pelo escritório Caique Castro Advocacia.

Arquitetura
src/app.py → cria janela PyWebView com js_api=WebViewApi(...)
Frontend chama window.pywebview.api.<método>()
src/backend/webview_api.py → valida payload, abre dialogs nativos, orquestra serviços
src/services/<Módulo>/ → backend + frontend por serviço
Build → PyInstaller gera LEGAL-HUB.exe (OneDir)
Módulos Ativos
IA_PET_INICIAL Ativo

Analisa até 6 PDFs e gera petições iniciais completas em DOCX via OpenAI. Redução de ~70% no tempo de elaboração, gerando 50–100 documentos por ciclo.

IA_LAUDO Ativo

Analisa laudos médicos e periciais em PDF via OpenAI e retorna síntese estruturada para a equipe jurídica.

PDF_para_EXCEL Ativo

Extrai dados de PDFs via pdfplumber e exporta para planilhas Excel com openpyxl.

Analise_pastas Ativo

Preenche planilha Excel pela estrutura de pastas e renomeia arquivos em lote. Suporta dryRun antes de aplicar renomeações.

Gerador_lotes Ativo

Gera planilhas de integração por evento e consolidado a partir de templates configurados.

Relatorio_Conformidade Ativo

Gera documentos PRECA e RPV automaticamente a partir de planilha de entrada.

Sobre o Projeto

O Legal-Hub é uma aplicação desktop desenvolvida com PyWebView — Python no backend exposto como js_api e interface HTML/CSS/JS por serviço. A autenticação é feita via LDAP corporativo (ldap3), com fallback HTTP para desenvolvimento local.

O projeto foi desenvolvido a partir de novembro de 2025 com responsabilidade de ~70% do desenvolvimento total. Por ser um sistema interno em produção com dados sensíveis de clientes, o repositório é privado — mas uma demonstração em vídeo pode ser disponibilizada mediante solicitação.
