LLM Translate Extension 🔒 Privado

Extensão para Chrome, Brave e Edge que traduz textos selecionados, inputs e páginas inteiras 100% localmente usando modelos de linguagem rodando via Ollama — sem chaves de API, sem nuvem, sem rastreamento.

Instalar pela Chrome Web Store
Tech Stack
JavaScript Vanilla HTML5 & CSS3 Manifest V3 Ollama API Chrome Extensions API Web Speech API (TTS)
Sobre o Projeto

O LLM Translate Extension nasceu da necessidade de ter um tradutor que respeita a privacidade do usuário. Em vez de enviar o texto para servidores externos, a extensão se comunica com o Ollama rodando localmente — qualquer modelo de tradução compatível pode ser usado.

A arquitetura segue o Manifest V3 do Chrome, com scripts de conteúdo para injetar o tooltip flutuante nas páginas, service worker para orquestrar as chamadas à API do Ollama e um popup completo para tradução direta e configurações. O sistema de tradução de páginas inteiras usa concorrência limitada para processar blocos em paralelo sem travar o navegador.

Funcionalidades

Tradução offline e privada — todo o processamento acontece no seu computador. Sem custos de API, sem dados enviados para nuvem.

Botão flutuante de seleção — selecione qualquer texto na página e um ícone aparece ao lado. Clique para traduzir instantaneamente com opções de cópia e áudio (TTS).

Tradução progressiva de páginas — traduz páginas inteiras preservando o HTML original. Processamento em lotes paralelos com barra de progresso nativa e botão para restaurar o texto.

TTS integrado — ouça a pronúncia do texto traduzido usando o sintetizador de voz nativo do navegador.

Ajustes avançados — escolha o modelo Ollama instalado, idioma alvo, URL do servidor e customize o System Prompt da LLM.

Como Instalar e Rodar
Recomendado: GPU dedicada (NVIDIA, AMD ou Apple Silicon) com VRAM suficiente para o modelo escolhido. Rodar só em CPU pode deixar as traduções lentas.
1Instalar o Ollama

Baixe e instale o Ollama em ollama.com, depois baixe um modelo de tradução pelo terminal:

  • ollama pull zongwei/gemma3-translator:1b — ~880 MB, rápido e especializado (recomendado)
  • ollama pull translategemma:4b — modelo oficial do Google para tradução
  • ollama pull qwen3.5:0.8b — super leve, ideal para PCs mais modestos
2Iniciar o Ollama com CORS habilitado

A extensão precisa que o Ollama aceite conexões do navegador. Inicie com:

  • Windows: execute o arquivo iniciar_ollama.bat incluso no repositório
  • macOS / Linux: no terminal: OLLAMA_ORIGINS="*" ollama serve
  • PowerShell (alternativa): $env:OLLAMA_ORIGINS="chrome-extension://*" — permite apenas conexões de extensões, sem abrir para qualquer origem
3Instalar a extensão

Duas opções:

  • Chrome Web Store (mais fácil): clique no botão "Instalar pela Chrome Web Store" acima
  • Manual: acesse chrome://extensions/, ative o Modo do Desenvolvedor, clique em "Carregar sem compactação" e selecione a pasta local_translate/
4Configurar na primeira abertura

Clique no ícone da extensão na barra do navegador:

  • Verifique se o indicador no topo direito está Online (verde)
  • Vá em Ajustes → selecione o modelo Ollama instalado → defina o idioma alvo → Salvar
