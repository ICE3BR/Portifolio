Sobre o Projeto

O repositório chatbot-IA apresenta o ICEBERG IA, um chatbot interativo que utiliza o modelo de Inteligência Artificial DeepSeek-R1:8b executado na plataforma Ollama.

Este chatbot é projetado para processar linguagem natural de forma eficiente, oferecendo funcionalidades como persistência do histórico de conversas, suporte a comandos interativos no terminal e uma interface otimizada para uso eficiente de recursos.
