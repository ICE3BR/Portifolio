Calculadora de Benefícios Previdenciários 🔒 Privado

Aplicação web full stack para liquidação de sentenças previdenciárias, inspirada no Previdenciarista. Calcula benefícios atrasados com correção monetária INPC, juros de mora conforme Manual da Justiça Federal, prescrição quinquenal, 13º salário proporcional e abatimentos. Geração de PDF profissional e API REST documentada com Swagger. Deploy em produção com Nginx + Systemd no Linux.

Tech Stack
Python FastAPI Pandas FPDF2 Uvicorn HTML/CSS/JS Nginx Linux Systemd Swagger
Benefícios Suportados
Auxílio-Acidente 50% do salário de benefício
Auxílio-Doença 91% do salário de benefício
Aposentadorias 100% do salário de benefício
Funcionalidades

Cálculo completo de benefícios atrasados desde a Data de Início do Benefício (DIB), com reajustes anuais automáticos conforme índices do INSS.

Prescrição quinquenal — considera apenas os 5 anos anteriores à data de ajuizamento.

Correção monetária INPC/IBGE aplicada mensalmente sobre o período de competência.

Juros de mora conforme Manual da Justiça Federal — 1% a.m. até 06/2009, 0,5% a.m. até 04/2012, 0,4% a.m. (poupança) até 11/2021, 0% (SELIC) a partir de 12/2021.

13º salário proporcional calculado automaticamente por competência.

Abatimentos dinâmicos — suporte a múltiplos valores já pagos administrativamente.

Geração de PDF profissional com memória de cálculo detalhada pronta para juntada processual.

API REST documentada com Swagger em /docs — endpoints bem definidos e testáveis.

Infraestrutura de Produção
Nginx (reverse proxy — porta 80) FastAPI + Uvicorn (porta 8000) Servidor HTTP Python (frontend — porta 8080) Systemd (dois serviços gerenciados) Linux / Ubuntu
Sobre o Projeto

Inspirada no Previdenciarista, esta aplicação automatiza a liquidação de sentenças previdenciárias — um processo que antes era realizado manualmente em planilhas e sujeito a erros de atualização monetária e de cálculo de juros.

O backend em Python + FastAPI expõe uma REST API que aplica os índices INPC oficiais armazenados localmente em src/data/indices/. O frontend em HTML/CSS/JS consome a API e exibe os resultados em tabela detalhada. O deploy em produção roda com dois serviços Systemd (API e frontend) atrás do Nginx.

Repositório privado — sistema em produção com dados sensíveis de clientes. Para solicitar uma demonstração ou mais detalhes, entre em contato.
